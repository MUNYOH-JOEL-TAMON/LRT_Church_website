# Latter Rain Tabernacle (LRT) - Church Management Platform

## Overview
Latter Rain Tabernacle (LRT) is a comprehensive, enterprise-grade Church Management Platform. It is not just a standard website but a complete digital solution that includes a public-facing website, a member portal, and an administrative dashboard to manage church resources, events, sermons, and more.

## Tech Stack
- **Frontend:** React, TypeScript, Vite, React Router, Tailwind CSS, Axios
- **Backend:** Node.js, Express.js, TypeScript
- **Database:** MongoDB Community Server (Local Database) with Mongoose
- **Authentication:** JWT Authentication, Bcrypt
- **Architecture:** Modular Monolith

## Architecture Structure
The application follows a clean architecture (Modular Monolith) with strict separation of concerns:
`Routes -> Controllers -> Services -> Models -> Database`

## Development Roadmap
The project follows a strict 7-phase roadmap:
1. **Phase 1:** Project Planning
2. **Phase 2:** System Analysis
3. **Phase 3:** System Design
4. **Phase 4:** Backend Development
5. **Phase 5:** Frontend Development
6. **Phase 6:** Testing
7. **Phase 7:** Deployment Preparation
